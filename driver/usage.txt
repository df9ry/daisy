Usage of the daisy module
=========================

1. Loading without parameter will bring up the SPI chip on /dev/spidev0.0.

	$ sudo insmod daisy.ko
	
2. Loading with parameters will bring up another SPI chip or more than
   one SPI chip. There must be no spaces in the parameter string.

	$ sudo insmod spi_dev="/dev/spidev0.1"
	$ sudo insmod spi_dev="/dev/spidev0.0,/dev/spidev0.1"
	
3. Before you start the SPI interface you have to configure it.
   First you have to assign a HW address.

	$ sudo ifconfig dsy0 hw ether 12:34:56:78:9a:bc
	
4. After configuration you can start up operation by bringing the 
   interface up.

	$ sudo ifconfig dsy0 up
